# Startup Pitch Deck Analysis

This project is a web application that provides AI-powered analysis of startup pitch decks. Users can upload a pitch deck (in PDF format) or a video pitch to receive insights and scoring.

## Features

- **Pitch Deck Analysis**: Upload a PDF pitch deck to extract key information and receive AI-powered scoring.
- **Video Pitch Analysis**: Upload a pitch video for transcription and investor-style critique.
- **Market Research**: Get market size estimates and competitive landscape analysis.
- **GitHub Repository Analysis**: Analyze repository health and development activity for developer tools.
- **Q&A Assistant**: Ask questions about analyzed startups using our AI chatbot.

## Tech Stack

- [Next.js](https://nextjs.org/)
- [React](https://reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Shadcn UI](https://ui.shadcn.com/)

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
